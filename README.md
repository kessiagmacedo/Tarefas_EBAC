## 📝 Descrição das Tarefas

### 📂 Módulo 1 - Introdução
Primeiros passos com Python e Jupyter Notebooks.

### 📂 Análise de Dados
- **Arquivo:** `M10_tarefa_ebac.py`
- **Dataset:** E-commerce
- **Conteúdo:** Visualização de dados com Python

### 📂 Tarefa 13 - Projeto Supermercado
- **Dataset:** `MODULO7_PROJETOFINAL_BASE_SUPERMERCADO.csv`
- **Conteúdo:** Análise exploratória de dados de vendas

### 📂 Tarefa 14 - Tratamento de Dados
- **Dataset:** `CHURN_TELECON_MOD08_TAREFA.csv`
- **Conteúdo:** 
  - Identificação de valores nulos
  - Tratamento de dados faltantes
  - Padronização de variáveis
  - Limpeza de dados

### 📂 Tarefa 15 - Análise Exploratória (Churn)
- **Dataset:** `CHURN_TELECON_MOD15_TAREFA15_2_parte.csv` (dados tratados)
- **Conteúdo:**
  - Análise univariada e bivariada
  - Identificação de outliers
  - Correlação entre variáveis
  - Insights sobre Churn de clientes

#### 📊 Principais Insights:
| Descoberta | Insight |
|------------|---------|
| Idosos | 38,8% de churn vs 23,5% não-idosos |
| Contrato Mensal | 42,4% de churn (maior risco) |
| Contrato 2 anos | Apenas 1,5% de churn |
| Fibra Ótica | 42% de churn (insatisfação?) |
